❓ How does it work?

1️⃣ Create your questions
In the tab 📝 Create/Edit, write a statement (true or false), its explanation, then add between 2 and 4 sources. For each source, indicate whether it is reliable or unreliable — that's what the player will have to guess!

You can also attach an image or a PDF as a source using the 📎 button.
2️⃣ Test your set
Your questions build up in 📋 My questions. You can edit them, delete them, or add more.

Use the tab 🎮 Play to test your set before publishing it.
3️⃣ Publish in the Community
When your set is ready, go to 📤 Import / Export and click on 🌍 Publish to the Community.

Give your set a title, a short description, your nickname (optional), and the difficulty level. Accept the publication guidelines, then click Publish!

Your set will immediately be visible to all players in the section 🌍 Community.
💡 Good to know
• Your questions are saved in your browser — if you change devices, export them as JSON first.
• Once published, a set cannot be modified (export it first, edit it, then republish it).
• Content that does not comply with the guidelines may be removed without notice.

📋 How to use the Excel template

📂 What the file contains

The file has two tabs:

• Questions — fill in your questions here (one row = one question).

• Instructions — field reference if you're unsure about anything.

✏️ What to fill in

For each row (= question):

• Statement — the claim the player will have to fact-check.

• True or False — dropdown: TRUE or FALSE.

• Explanation — the answer revealed at the end of the question.

• Nuanced topic? — YES if the topic is debated (no clear consensus).

For each source (Src1 → Src4):

• Name — the name of the source.

• Reliable? — dropdown: YES (reliable) or NO (unreliable). This is what the player has to guess!

• Description & Quote — optional but recommended.

• Emoji — optional, one single emoji to visually represent the source.

• Feedback — message shown depending on whether the player guessed correctly.

➕ How to add more statements

Simply add a new row in the "Questions" tab. Each row is an independent question. The first data row is the pre-filled example — you can edit or delete it.

Src3 and Src4 are optional — leave them empty if you only have 2 sources.

📥 How to import it afterwards

1. Save your Excel file (.xlsx).

2. Come back here in FakeMeter, tab Import / Export.

3. Click 📥 Import filled template.

4. Select your file — your questions are added automatically!
