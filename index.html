Télécharge le modèle Excel (.xlsx), remplis-le (Excel, Google Sheets, LibreOffice…), puis réimporte-le ici.

❓ Comment ça marche ?

1️⃣ Crée tes questions
Dans l'onglet 📝 Créer/Modifier, rédige une affirmation (vraie ou fausse), son explication, puis ajoute entre 2 et 4 sources. Pour chaque source, indique si elle est fiable ou non fiable — c'est ce que le joueur devra deviner !

Tu peux aussi joindre une image ou un PDF comme source via le bouton 📎.
2️⃣ Teste ton set
Tes questions s'accumulent dans 📋 Mes questions. Tu peux les modifier, les supprimer, ou en ajouter d'autres.

Utilise l'onglet 🎮 Jouer pour tester ton set avant de le publier.
3️⃣ Publie dans la Communauté
Quand ton set est prêt, va dans 📤 Importer / Exporter et clique sur 🌍 Publier sur la Communauté.

Donne un titre à ton set, une courte description, ton pseudo (optionnel) et le niveau de difficulté. Accepte la charte de publication, puis clique sur Publier !

Ton set sera immédiatement visible par tous les joueurs dans la section 🌍 Communauté.
💡 Bons à savoir
• Tes questions sont sauvegardées dans ton navigateur — si tu changes d'appareil, exporte-les en JSON d'abord.
• Une fois publié, un set ne peut pas être modifié (exporte d'abord, modifie, puis republie).
• Les contenus ne respectant pas la charte peuvent être supprimés sans préavis.

📋 Comment utiliser le modèle Excel

📂 Ce que contient le fichier

Le fichier a deux onglets :

• Questions — tu remplis tes questions ici (une ligne = une question).

• Mode d'emploi — rappel de chaque champ si tu as un doute.

✏️ Ce qu'il faut remplir

Pour chaque ligne (= question) :

• Affirmation — la phrase que le joueur devra vérifier.

• Vrai ou Faux — menu déroulant : VRAI ou FAUX.

• Explication — la réponse révélée à la fin de la question.

• Sujet nuancé ? — OUI si le sujet est controversé (sans consensus clair).

Pour chaque source (Src1 → Src4), tu remplis :

• Nom — le nom de la source.

• Fiable ? — menu déroulant : OUI (fiable) ou NON (peu fiable). C'est ce que le joueur devra deviner !

• Description & Citation — optionnels, mais recommandés.

• Emoji — optionnel, un seul emoji pour représenter la source visuellement.

• Feedback — message affiché selon si le joueur a bien deviné ou non.

➕ Comment ajouter des affirmations

Ajoute simplement une nouvelle ligne dans l'onglet "Questions". Chaque ligne est une question indépendante. La première ligne de la feuille (Src1 + Src2) correspond à l'exemple déjà rempli — tu peux la modifier ou la supprimer.

Src3 et Src4 sont facultatives — laisse-les vides si tu n'as que 2 sources.

📥 Comment importer ensuite

1. Sauvegarde ton fichier Excel (.xlsx).

2. Reviens ici dans FakeMètre, onglet Importer / Exporter.

3. Clique sur 📥 Importer le modèle rempli.

4. Sélectionne ton fichier — tes questions s'ajoutent automatiquement !
